    Things to note:

    • macros can be in same file or included from separate file
    • macros allow you to make a change in one location and have the change reflected everywhere else in the program
    • macros are more useful with things that are used repeatedly
    • read more at http://en.wikipedia.org/wiki/C_preprocessor (Section Macro Definition and Expansion
    • read more at http://gcc.gnu.org/onlinedocs/cpp/Macros.html if you want to learn more advance uses of macros

  • Assignment

    write a program that includes a macro file (similar to common_constants.h)

    • the macro file will define the value for PI and a function to calculate the area of a square
    • the program will ask for a length and then calculate the following:
      • circumference of a circle with a diameter equal to the inputted length
      • area of a circle with a diameter equal to the inputted length
      • area of a square with a side equal to the inputted length
    • if 1 was the inputted length, the program will use macros to print out the following: 
      the circumference of the circle is: 3.14
the area of the circle is: 6.28
the area of the square is: 1

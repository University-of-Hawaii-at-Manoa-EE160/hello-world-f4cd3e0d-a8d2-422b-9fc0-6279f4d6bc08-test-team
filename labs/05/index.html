  • Character Analyzer

      Create a program (character_analyzer.c) that will:
    • analyze each character you type in
    • tell you if you've entered a upper case letter
    • tell you if you've entered a lower case letter
    • tell you if you've entered a special character (don't worry about DEL or any other ASCII characters not on the keyboard)
    • quit the program if you've entered q

    Your program should have if/else statements nested inside a while loop

  • Number Repeater

      Create a program (number_repeater.c) that will:
    • repeat the number you type in a specific number of times
    • if you enter 0 thru 10, the program will repeat the number by the same amount
    • if you enter 11 thru 100, the program will repeat the number 15 times
    • if you enter anything greater than 100, the program will repeat the number 20 times
    • if you enter 999, the program will exit
      Your program should have:
    • a main while loop
    • if/else statements inside the main while loop
    • nested while loop inside the if/else statements

    write a program that asks the user for a number until the user gives a number that is greater than 100

  • Summer

    write a program that:

    • asks the user for a number
    • if the number is greater than 100, then the program will output the number less 5 three times
    • if the number is less than or equal to 100, then the program will output the number to the third power

    example outputs:

    • if the user enters 101, then the outputs will be 96, 91, and 86
    • if the user enters 90, then the output will be 729000

  • Summer that sums 10

      Create a program (summer_10.c) that will
    • accept 10 decimal numbers
    • print out a message notifying the user that it is the last decimal number before the program accepts the last decimal number
    • sum up the 10 decimal numbers
    • print out the sum of the 10 decimal numbers

  • Summer that terminates with 999

      Create a program (summer_999.c) that will:
    • accept decimal numbers until the user enters 999
    • sum up the decimal numbers
    • print out the sum of the decimal numbers
    • the 999 value should not be included in the outputted sum

  • Summer that user specifies

      Create a program (summer_user.c) that:
    • asks the user for the number of decimal numbers s/he will be entering
    • accepts the number of decimal numbers specified by the user
    • print out a message notifying the user that it is the last decimal number before the program accepts the last decimal number
    • sum up the decimal numbers
    • print out the sum of the decimal numbers

  • Summer that continues until

      Create a program (summer_until.c) that:
    • asks the user if there's more numbers to be entered
    • accepts more decimal numbers until user says there are no more decimal numbers to be entered
    • sum up the decimal numbers
    • print out the sum of the decimal numbers
